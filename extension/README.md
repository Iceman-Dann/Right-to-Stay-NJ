# 🏠 Right to Stay NJ — Chrome Extension

An AI-powered browser extension that helps New Jersey tenants understand their rights instantly.

---

## 📦 How to Install (Developer Mode)

1. Make sure your local server is running by running `npm run dev` and `npx tsx scripts/local-backend.ts` in the main project folder.
2. **Open Chrome** and navigate to `chrome://extensions`
3. Enable **Developer Mode** (toggle in the top right)
4. Click **"Load unpacked"**
5. Select the `extension/` folder inside this project directory
6. The extension icon will appear in your Chrome toolbar. It is completely plug-and-play and automatically routes requests to your local server backend!

---

## ✨ Features

| Feature | Description |
|---|---|
| 💬 **AI Chat** | Ask any NJ tenant rights question, get instant plain-English answers |
| 📄 **Notice Explainer** | Paste any eviction notice text to get it decoded in plain language |
| 🖱️ **Right-Click Explain** | Select any text on any webpage and right-click → "Explain with Right to Stay NJ" |
| 🌐 **Bilingual** | Fully English/Spanish — toggle with EN/ES buttons |
