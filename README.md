## ⚖️ The Problem: A Stark Legal Imbalance

In New Jersey's Special Civil Part (Landlord-Tenant) courts, the scales of justice are heavily tilted against renters:
* **80%+ of landlords** enter hearings represented by corporate lawyers.
* **90%+ of tenants** stand completely alone (*pro se*), struggling to decipher court summonses and complex filing deadlines.
* **The Access Gap**: While New Jersey has some of the strongest tenant protection laws in the United States, they are meaningless if tenants do not know how or when to assert them. A minor technical mistake or a misleading settlement in a drafty court hallway often leads to immediate displacement.

---

## 🛡️ The Shield: Our Solution

**Right to Stay NJ** is a completely free, open-source, privacy-first legal-tech portal designed to give tenants the tools to defend their homes. By translating intimidating court procedures into visual guides and deploying localized AI calculation engines, we level the playing field.

### Core Interactive Features

* **📄 Visual Summon Annotation Decoder**: Decodes intimidating official state paperwork (such as *Notice to Quit* and *Summons & Complaint*) to explain what key clauses actually mean.
* **⚖️ AI Legal Defense Auditor & Package Generator**: Automatically evaluates rent payments, security deposits, and habitability defects, outputting a formal Special Civil Part Answer packet.
* **🎭 AI Court Mediation Simulator**: Lets tenants practice negotiating against a mock aggressive landlord attorney in a safe, simulated chat environment to avoid eviction traps.
* **✍️ Landlord Letter Center**: Generates formal letters in English and Spanish for:
  - *Habitability Deficiencies* (under *Marini v. Ireland* rules).
  - *Cease & Desist Harassment* (warning landlords of criminal lockouts under **N.J.S.A. 2A:39-1**).
  - *Security Deposit Returns* (calculating double-damages eligibility under **N.J.S.A. 46:8-21.1**).
  - *Winter Moratorium Disputes* (verifying Board of Public Utilities energy moratoria).
* **🌡️ Daily Temperature & Habitability Log**: Tracks day/night temperatures and vital service failures inside the apartment, exporting a print-ready evidence grid to hand directly to the judge.
* **🏢 County Welfare & Emergency Aid Finder**: Connects tenants instantly to their county's specific Emergency Rental Assistance Program (ERAP), courthouse locator, and legal services.

---

## 🔒 Privacy & Abuse Protections

* **Privacy-First Design**: All calculators, logs, and defense packets are processed and saved locally in the browser’s `localStorage`. We do not log names, addresses, or lease files in any database, ensuring 100% tenant confidentiality.
* **IP-based Rate Limiter**: The serverless `/api/ai-chat` endpoint restricts requests to a maximum of **15 messages per 10 minutes** per client IP, securing your API billing keys against scraper bots.

## 🛠️ Tech Stack

* **Core Framework**: React 18 with TypeScript 5
* **Styling**: Vanilla CSS with custom theme design system (dark mode support, active tap physics, CSS variable overrides, and print media query stylesheets)
* **Build System**: Vite 7
* **AI Provider**: Groq Cloud running **Llama 3.1 8B Instant** (with absolute safety-override filters for New Jersey landlord laws)
* **Extensions**: Chrome manifest v3 extension supporting active page analysis

## ⚡ Quickstart

### 1. Configure API Key
Create a `.env` file in the root directory:
```bash
GROQ_API_KEY=your_groq_api_key_here
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
Start the frontend dev server:
```bash
npm run dev
```

In a separate terminal window, start the local backend server (which proxies API requests):
```bash
npx tsx scripts/local-backend.ts
```

### 4. Build for Production
Compiles and bundles static files:
```bash
npm run build
```
